== Description ==
 
Help to add all Egypt Cities/States for WooCommerce Check Out, And Setup to be transleted With WPML Plugin .
 


== Installation ==
  
1. Upload `plugin file` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
2- Will see two dropdown in checkout page for Cities and States


 
== Frequently Asked Questions ==
 
= Are there all Cities and region in Egypt ? = 
it's almost contain for all Cities and States in Egypt .


= Is it translatable ? =  
yes it's can be translated with wpml plugin .
